A HTML5 canvas game based on Snake VS Block mobile game. [Try (on mobile only)](https://snake-vs-block.now.sh)
=====================

# Setup
```bash
git clone https://github.com/XavierLeTohic/snake-vs-block
npm install
```

## Run in development
```bash
npm run dev
```

## Run in production
```bash
npm run build && npm run start
```
